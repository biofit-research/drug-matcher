데이터 저장·이용 안내

업로드한 본원 약품 마스터·HIRA 매핑 데이터는 이 브라우저(localStorage)에만 저장되며, GitHub·외부 서버로 전송되지 않습니다.

  • 장치·브라우저별 분리 — 다른 PC, 휴대폰, 또는 같은 PC의 다른 브라우저(Chrome/Edge/Safari)에서는 다시 업로드해야 합니다
  • 시크릿/프라이빗 모드 X — 창을 닫으면 데이터가 사라집니다
  • 브라우저 기록 삭제 시 사라짐 — "쿠키 및 사이트 데이터 삭제" 시 함께 제거됩니다
  • 저장 용량 약 5–10MB — HIRA 14,000행 + 본원 마스터까지 올리면 한도에 가까워질 수 있어 한도 초과 시 안내가 표시됩니다
  • 여러 사용자 데이터 공유 불가 — 정적 호스팅(GitHub Pages) 특성상 같은 데이터를 모든 사용자에게 동일하게 보여주려면 별도 백엔드가 필요합니다
